# Lunzheng Li - Personal Academic Website

Personal website for Lunzheng Li, researcher in behavioral economics, experimental economics, and economics of science.

**Live site:** [lunzhengli.site](https://lunzhengli.site/)

## About This Repository

This website was rebuilt using [Hugo](https://gohugo.io/) static site generator with the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme. The migration from raw HTML to Hugo was done for easier content management and maintenance.

### Current Stack

- **Framework:** Hugo (v0.146.0+)
- **Theme:** PaperMod with custom styling
- **Customizations:**
  - Custom header with search and contact icons
  - Expandable abstracts for research papers
  - Card-based layout for research and teaching pages
  - Two-column contact page layout
  - Self-hosted Fira Code font
